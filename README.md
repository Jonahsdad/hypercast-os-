# Hypercast OS

Hypercast OS is a modular forecasting engine ecosystem.

It provides:

- **Backend API** (FastAPI) for managing forecast targets and forecasts.
- **Core Engine** for time-series forecasting and evaluation.
- **Streamlit Frontend** for quick exploration and interaction.
